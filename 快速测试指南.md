# 快速测试指南

## 1. 启动服务

### 方法一：使用启动脚本（推荐）
```bash
cd /Users/liting/youtube-scraper
./start.sh
```

### 方法二：手动启动
```bash
cd /Users/liting/youtube-scraper/backend
npm start
```

看到以下提示表示启动成功：
```
后端服务运行在 http://localhost:3001
```

## 2. 打开前端页面

在浏览器中打开：
```
file:///Users/liting/youtube-scraper/frontend/public/index.html
```

## 3. 测试邮件发送功能

### 步骤 1：生成测试 Excel 文件
```bash
cd /Users/liting/youtube-scraper
python3 generate_test_excel.py
```

这会生成一个包含 8 个测试客户的 Excel 文件。

### 步骤 2：切换到邮件发送标签
- 点击页面顶部的「邮件发送」标签

### 步骤 3：配置 SMTP（以 Gmail 为例）
如果你有 Gmail 账号：
1. 发件邮箱：your@gmail.com
2. SMTP服务器：smtp.gmail.com
3. 端口：587
4. 密码：使用 Gmail 应用专用密码

**获取 Gmail 应用专用密码**：
1. 访问 https://myaccount.google.com/security
2. 开启「两步验证」
3. 搜索「应用专用密码」
4. 生成一个新密码并复制到密码输入框

点击「测试连接」按钮，应该显示「SMTP连接测试成功」。

### 步骤 4：编辑邮件内容
邮件标题：```
合作邀请：{客户名称}您好！
```

邮件正文：```
尊敬的 {客户名称}，您好！

这是一封测试邮件，验证邮件发送功能是否正常。

如有问题，请联系我们。

祝好！
```

发送间隔：2 秒

### 步骤 5：上传 Excel 文件
1. 点击上传区域
2. 选择刚刚生成的「客户列表模板.xlsx」文件
3. 上传成功后会显示 8 个客户

### 步骤 6：发送邮件
1. 确认已勾选所有客户（默认全选）
2. 点击「📧 发送邮件」按钮
3. 观察发送进度

### 步骤 7：查看结果
发送完成后会显示：
- 总数：8
- 已发送：8
- 成功：8
- 失败：0

## 4. 测试模板功能

### 保存模板
1. 编辑邮件内容
2. 点击「保存模板」
3. 输入模板名称，如：「合作邀请模板」
4. 模板会保存在浏览器本地存储中

### 加载模板
1. 从下拉菜单选择「合作邀请模板」
2. 邮件内容会自动填充

### 删除模板
1. 从下拉菜单选择要删除的模板
2. 点击「删除模板」

## 5. 测试失败重试功能

### 模拟失败场景
1. 在 Excel 中添加一个无效邮箱：`invalid-email`
2. 重新上传并发送
3. 会看到部分失败的记录

### 重试失败邮件
1. 点击「🔄 重试失败邮件」按钮
2. 系统会重新发送失败的邮件

## 常见问题排查

### 问题：后端无法启动
**解决方案**：
```bash
cd /Users/liting/youtube-scraper/backend
rm -rf node_modules
npm install
```

### 问题：SMTP 连接失败
**检查清单**：
- [ ] 确认 SMTP 服务器地址正确
- [ ] 确认端口号正确（587 或 465）
- [ ] 确认使用了授权码而非密码（QQ/163邮箱）
- [ ] 确认已开启 SMTP 服务（登录邮箱设置中开启）

### 问题：Excel 上传失败
**检查清单**：
- [ ] 文件格式必须是 .xlsx（不是 .xls）
- [ ] 文件大小不超过 10MB
- [ ] Excel 中包含「客户名称」和「邮箱地址」列

### 问题：邮件发送失败
**常见原因**：
- 邮箱地址格式错误
- SMTP 配置错误
- 发送频率过快被限制
